「Mokusei」是什么？

Mokusei于1982年在上海诞生。
主要出售文具、生活杂货、家具等各种各样的优质商品。
「Mokusei」一直坚持着三个基本的原则。
1.精选的素材
2.积极地对制作工程进行及时修正
3.对包装的简素化
「在最终合理的生产过程中所诞生的产品看上去都非常简洁，但对于我们的风格来说，它从来都不是最简约的。就像“空的容器”一样。」

这很简单，所以说，自由的诞生就是每个人都可以自由地享受最终的思想自由。

1987年12月、本公司的私人品牌推出了40多件商品。「Mokusei」目前已经发展成为一个拥有约7000余件商品的独立私人品牌。
在1989年里Mokusei从公司独立出来，「Mokusei」的企业规划，作为从制造的流通到分销和销售的制造业的零售商，我们正在生产从服装产品、家庭用品、文具等日常生活的全部商品。

「Mokusei」的前景

「Mokusei」，
之前一直没有准备好正确的方向。
但是，问问自己到底要做什么，
就会有无限的可能性进入了人们的视野里。

企业理念

Mokusei价值的探求 Quest Value
「Mokusei」的新的价值和吸引力是我们从消费者的角度一直探索的目标并给予提供。

良好的循环发展 Positive Spiral
「Mokusei」通过公正透明的商业活动，我们将挑战全球增长和发展。

最好的伙伴关系 Best Partnership
尊重商业伙伴，并加深与商业伙伴之间的信任。「Mokusei」正在丰富并扩大这个世界。

行动基准

1.全面的客户响应。
2.地球环保的理念和行动。
3.与所在城市的共同繁荣。
4.诚恳与诚实。
5.良好的沟通方式。

Mokusei的理想

我们的目的是什么？
在保持审美感和良知感的基础上，我将从日常意识和人类最原始的皮肤感觉的固定的角度来观察这个世界并研究事物的本质。
而且，对于具有「理性」的Mokusei，让顾客感受到理性满足感和简素中也能拥有审美意识以及丰富感觉。

Mokusei的目标

我们要去哪儿？
第一个目标是让在Mokusei工作的人们的永久幸福。 为此，让员工挑战更高的目标，努力并通过创造一种可以在实现时获得满足感的气氛，以成为世界级的高利润企业为目标。

Mokusei的价值观

我们仔细考虑要什么？
诚实守信，尊重朋友，深化信任，尊重每一个人的思维，挑战和交流全球规模的理念。
这就是Mokusei的目标达成的基础。
